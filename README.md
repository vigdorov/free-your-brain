### Скрипты

```bash
# Установка\Обновление зависимостей
npm run init

# Запуск приложения с открытием страницы
npm start

# Запуск приложения без открытия страницы
npm run dev

# Сборка приложения
npm run build

# Проверка кода линтером и проверка тайпингов
npm run lint

# Запуск тестов
npm run test
```
